# microservice-template
A template for developing a microservice

# Installation
* `npm install`

# Run
* Make sure you have a RabbitMQ broker running.
* Set environment variables like `MONGO_URL` and `RABBITMQ_URL`. Full list in the wiki.
* `npm start`

# Test
`npm test`
